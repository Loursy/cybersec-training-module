# 🛡️ CyberSec Training Module - Web API Security & Awareness

CyberSec Training Module, siber güvenlik farkındalığını artırmak ve geliştiricilere güvenli kodlama pratiklerini öğretmek amacıyla tasarlanmış, uygulamalı (hands-on) bir eğitim platformudur. Özellikle Web API güvenliği üzerine odaklanan bu modül, teorik bilgileri pratik zafiyet senaryolarıyla birleştirir.

## ✨ Öne Çıkan Özellikler

* **Hands-On Training Model:** Kullanıcıların sadece pasif birer okuyucu olmak yerine, gerçek dünya web zafiyetlerini simüle edilmiş güvenli bir ortamda bizzat deneyimleyerek öğrenmesini sağlayan interaktif altyapı.
* **Web API Zafiyet Simülasyonları:** İstemci tarafı (client-side) manipülasyonlarından, arka plan veritabanı (database) mantıksal hatalarına kadar çeşitli atak vektörlerinin incelenebileceği senaryolar.
* **Aktif Keşif ve Analiz:** Eğitim modülleri, kullanıcıların tarayıcı geliştirici araçlarını (DevTools / F12) aktif olarak kullanmasını, sayfa kaynağını incelemesini ve API uç noktalarındaki zafiyetleri keşfetmesini gerektirecek şekilde tasarlanmıştır.
* **Taşınabilirlik (Portability):** Herhangi bir harici veritabanı sunucusu kurulumuna gerek kalmadan, SQLite sayesinde anında çalıştırılabilir tak-çalıştır mimari.

## 🚀 Teknolojiler ve Araçlar

* **Mimari:** RESTful Web API
* **Backend:** Node.js, Express.js (Zafiyet simülasyonları için native JavaScript tercih edilmiştir)
* **Veritabanı:** SQLite
* **Güvenlik Test & Analiz Araçları:** Browser DevTools, Postman / Insomnia
