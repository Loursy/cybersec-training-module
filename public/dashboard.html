Siber Güvenlik API Eğitim Platformu

Eğitim Modüllerine Hoş Geldiniz

Lütfen test etmek ve simüle etmek istediğiniz zafiyet modülünü seçin.

1. SQL Injection ✔

Kullanıcı girdilerinin filtrelenmediği durumlarda veritabanı sorgularının nasıl manipüle edildiğini öğrenin ve test edin.

2. Broken Access Control

Yetkilendirme kontrollerinin atlanarak, başka kullanıcılara veya admin paneline nasıl erişilebileceğini deneyimleyin.

3. Excessive Data Exposure

API yanıtlarında gereğinden fazla veri döndürülmesinin yarattığı bilgi sızıntısı risklerini analiz edin.

4. Cross-Site Scripting (XSS)

Zararlı JavaScript kodlarının sisteme enjekte edilerek diğer kullanıcıların tarayıcısında nasıl çalıştırıldığını görün.

5. Rate Limiting

API uç noktalarına sınır konulmadığında Brute Force (Kaba Kuvvet) saldırılarının nasıl yapıldığını simüle edin.

6. OS Command Injection

Sistem terminaline doğrudan erişim sağlayan komut enjeksiyonu zafiyetinin kritik etkilerini laboratuvar ortamında test edin.
